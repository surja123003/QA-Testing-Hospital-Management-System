# Hospital Management System - Manual Testing Project

## Project Overview

This repository contains Manual Testing artifacts created for the Hospital Management System application.

The application manages:

* Patient Registration
* Doctor Management
* Appointment Scheduling
* Billing System
* Patient Records

* Regression Testing
* UI Testing
* End-to-End Testing
