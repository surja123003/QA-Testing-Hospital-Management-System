# Test Plan

## Project Name

Hospital Management System

## Objective

To verify the functionality, reliability, and usability of the Hospital Management System.

## Scope

* Login Module
* Patient Registration
* Doctor Management
* Appointment Scheduling
* Billing
* Search Patient
* Logout

## Testing Types

* Functional Testing
* Smoke Testing
* Sanity Testing
* Regression Testing
* UI Testing

## Test Environment

* OS: Windows 11
* Browser: Chrome
* Database: MySQL

## Entry Criteria

* Application deployed successfully
* Test environment available

## Exit Criteria

* All test cases executed
* Critical defects resolved

## Deliverables

* Test Cases
* Bug Report
* RTM
* Test Summary Report
